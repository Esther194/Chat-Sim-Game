# 碳權聊天室遊戲
## 簡介
聊天室外型的網頁遊戲，隨著劇情進展逐漸了解金融知識，並根據玩家不同的選擇開啟不同的劇情

## 使用方法
下載檔案(main.py、plot.xlsx、static資料夾、templates資料夾)

確認模組可使用

運行main.py

在瀏覽器開啟python控制台輸出的網址(http://127.0.0.1:5000)
